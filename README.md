## Features

- Trip type selection (one-way / hourly)
- Google Places Autocomplete for pickup, stops, and dropoff (US only)
- Distance and travel time via Google Distance Matrix API
- Phone number lookup with customer recognition
- Form validation with Zod
- Responsive design

## Tech Stack

React 18 · TypeScript · Vite · Tailwind CSS v4 · Zod · react-phone-number-input
· @react-google-maps/api

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Add your Google Maps API key

Create a `.env` file in the project root:

```
VITE_GOOGLE_MAPS_API_KEY=your_api_key_here
```

### 3. Run the dev server

```bash
npm run dev
```
